# anti-doomscroll

A small learning repo for replacing impulsive scrolling with one short, structured software-engineering lesson.

## What is inside
- A JavaScript and TypeScript curriculum split into short lessons
- Lessons designed to be useful even when your attention is low
- A consistent **T-KWICK** format so each page is easy to scan and start

## How to use it
1. Open the next lesson in `/home/runner/work/anti-doomscroll/anti-doomscroll/javascript-typescript-mentor`.
2. Read the **TL;DR** only.
3. Do the **one tiny action** at the end.

One small learning action is better than opening a feed.
